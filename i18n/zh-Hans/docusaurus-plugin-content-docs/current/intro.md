# 密码学培训课程

欢迎来到加密货币密码学培训课程！本课程专为应届生设计，将带你从零开始理解区块链背后的核心密码学技术。

## 课程目标

完成本课程后，你将能够：

- 理解对称加密与非对称加密的区别
- 掌握椭圆曲线密码学（ECC）的基本原理
- 深入理解 ECDSA 数字签名算法的工作机制
- 了解比特币/以太坊如何使用这些技术

## 课程特色

- **可视化教学**：嵌入 Desmos/GeoGebra 交互式图表，直观理解椭圆曲线
- **小数值示例**：使用简化的数字演示算法流程，便于手算验证
- **真实案例**：PS3 私钥泄露、比特币交易签名等实战案例

## 前置知识

- 基础编程能力（Python/JavaScript）
- 模运算概念（a mod n）
- 基础数论知识（素数、互质）
