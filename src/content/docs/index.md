# Cryptography Training Course

Welcome to the Cryptocurrency Cryptography Training Course! Designed for new graduates, this course takes you from zero to understanding the core cryptographic technologies behind blockchain.

## Course Objectives

After completing this course, you will be able to:

- Understand the difference between symmetric and asymmetric encryption
- Master the basic principles of Elliptic Curve Cryptography (ECC)
- Deeply understand how the ECDSA digital signature algorithm works
- Learn how Bitcoin and Ethereum use these technologies

## Course Features

- **Visual Learning**: Embedded interactive Desmos/GeoGebra charts for intuitive understanding of elliptic curves
- **Small Value Examples**: Simplified numerical demonstrations of algorithm flows for easy manual verification
- **Real-world Cases**: Practical cases like the PS3 private key leak and Bitcoin transaction signing

## Prerequisites

- Basic programming skills (Python/JavaScript)
- Concept of modular arithmetic (a mod n)
- Basic number theory (prime numbers, coprime)
